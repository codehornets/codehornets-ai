## 3 Most Common Commands

### 1. Formal Evaluation (100-point APEXX)
```
"Use marie-dance-evaluator to create a formal evaluation for Emma:
```

### 2. Quick Class Note
```
"Use marie-dance-evaluator to create a quick note for Emma:
- Good energy today
- Rock needs work
- Applied corrections well"
```

### 3. Batch (Multiple Students)
```
"Use marie-dance-evaluator to create evaluations for:
- Emma: [observations]
- Sophia: [observations]
- Maya: [observations]"
```
